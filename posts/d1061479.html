更换网页链接

默认情况下，hexo上传的博客链接形如:https://coder67.cn/2020/05/29/博客标题/ 这种形式。我们可以通过abbrlink插件将其更换为永久唯一的随机字符串的网址。形如
https://coder67.cn/8ddf18fb.html。
注：如果所有链接一样，请运行hexo clean
hexo-abbrlink

使用waline作为评论系统

请注意你的next主题版本：theme-next 或者 next-theme

首先我按照 https://www.npmjs.com/package/@waline/hexo-next 通过npm安装后，在配置文件更改waline会导致网站所有内容丢失。后面找waline作者讨论，发现了问题所在：

解决办法：

参考github链接修改next主题文件：https://github.com/theme-next/hexo-theme-next/pull/1582/files#
在config文件配置如下：

# Waline
# For more information: https://waline.js.org, https://github.com/lizheming/waline
waline:
  enable: true
  serverURL: https://vercel-kohl-seven.vercel.app # Waline server URL
  placeholder: Just go go # Comment box placeholder
  avatar: mm # Gravatar style
  guest_info: nick,mail,link # Custom comment header
  pageSize: 10 # Pagination size
  language: # Language, available values: en, zh-cn
  visitor: false # Article reading statistic
  highlight: false # Comment text highlight
  avatarForce: false # 注意，一定要这句话
  emojiCDN: # Emoji server URL
  emojiMaps: # Emoji map list
  comment_count: true # If false, comment count will only be displayed in post page, not in home page

番外

在waline评论区设置好后，还出现了另一个问题。当我开启评论计数和阅读次数时，这两部分均出现了2次。

由于前面通过npm安装了一次waline，现在需要通过 npm uninstall @waline/hexo-next把其删除。
